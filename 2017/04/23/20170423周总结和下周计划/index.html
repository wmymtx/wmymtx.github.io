这周工作总结（这周主要做了4件事）

  1. App Store上传审核，目前审核两次依然失败，主要是对审核要求不太清楚导致。

  2. 自主报表-数据表模块防数据访问慢的优化，经过三天的努力，由于没有源代码，造成定位问题比较困难，最初只能找到问题在服务端。最后经过分析日志，详细精确到某个环节乃至某个耗时的方法，并将结果反馈至公司产品组，正在优化。

  3. Andorid端经分简报展示优化已完成，本来计划这周上线，但遇到产品组网络问题，无法上传打包。

  4. DataOne服务端编译，现在已经将几个主要模块的代码编译通过

下周计划

  1. 继续完成上周未完成的工作

  2. 编译DataOne

  3. 初步翻阅架构师所涉书籍
